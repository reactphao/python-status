# python-status

Small Python project for status checks and experiments.

## Setup

No third-party dependencies yet.

```bash
python3 --version
```

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Run

```bash
python3 backend/day1.py
```
