# Makefile 简明介绍

## 1. Makefile 是什么

`Makefile` 是一个“命令集合”，用来把常用操作封装成短命令，减少重复输入。
你可以把它理解成“项目快捷键”。

## 2. 基本用法

在项目根目录执行：

```bash
make <目标名>
```

比如：

```bash
make setup
make run
```

## 3. 这个项目里的 Makefile 做了什么

当前 `Makefile` 提供 4 个目标：

- `make help`：列出可用命令
- `make setup`：创建虚拟环境并安装依赖
- `make run`：运行示例脚本
- `make clean`：删除虚拟环境和缓存

## 4. Makefile 的结构（你现在先认识即可）

格式是：

```
目标名:
	命令1
	命令2
```

注意：命令前面必须是 **Tab**，不是空格。

## 5. 常见问题

Q: 为什么用 `make`？
A: 让同学或未来的自己，用更短的命令完成同样的事，减少出错。

Q: `make` 一定要学吗？
A: 不必立刻学深，先会用就行。以后项目复杂了再深入。
